# 视频友好 HTML 主题库

本目录包含专为**视频演示**优化的 HTML 幻灯片主题。每个主题都是完整的单文件 HTML 模板，可直接复制并在浏览器中打开。

## 主题列表

### [Blue Professional](./blue-professional/)

奶油色纸张背景 + 电光钴蓝强调，干净现代的专业风格。

- 适合：B2B 讲解、投资报告、知识分享
- 特点：浅色背景、高对比、大字号、低密度

### [Purple Gold Presentation](./purple-gold-presentation/)

近黑电影感画布 + 紫色标题 + 金色强调，暗色教程风格。

- 适合：技术教程、代码 walkthrough、深夜沉浸感内容
- 特点：暗色背景、紫金配色、视频友好布局

### [BlockFrame](./blockframe/)

新粗野主义糖果色块 + 4px 粗黑边框 + 硬偏移阴影，高能量设计导向风格。

- 适合：创意提案、品牌重塑、设计评审、活泼教程
- 特点：米白/奶油主背景、粉蓝绿黄点缀、全 inline CSS/SVG、视频友好

### [BlockFrame Dark](./blockframe-dark/)

BlockFrame 的夜间模式：纯黑画布 + 白色粗边框 + 糖果色荧光光晕。

- 适合：暗色教程、视频分镜、创意提案、深夜沉浸感内容
- 特点：纯黑背景、高对比、10vw 安全边距、视频友好

## 使用方式

1. 进入对应主题目录，打开 `template.html` 预览效果。
2. 复制 `template.html` 作为起点，修改内容。
3. 参考 `design.md` 了解颜色、字体、组件规范。
4. 使用项目根目录的 `screenshot_html_slides.py` 导出 1920×1080 PNG。
